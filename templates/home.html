Online Learning Tool

Teaching yourself a new topic using an online learning environment doesn't have to be difficult! You can use this online learning tool to help yourself learn a new technical skill.

INFORMATION ABOUT THE TOOL

  1. Go through the learning materials
  2. Attempt the practicals
  3. Ask any questions you're unsure of on the discussion forum
  4. Attempt the assessment quiz to test your knowledge
