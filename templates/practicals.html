Practicals

Here, you will find all the practical exercises for learning this new technical module! Attempt each exercise as best as you can, and remember to use the discussion forum if you have any questions.

  • Practical 1, which should be attempted after watching Topic 1 and Topic 2
  • Practical 2, which should be attempted after watching Topic 3 and Topic 4
